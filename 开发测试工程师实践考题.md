##### 题目：

1. 用 Playwright  写一个小程序，自动化测试一下 baidu 首页 或 其他公开网站。1~3个 test cases 就好。
2. 做一个小网站，包括前后端，只要有登录功能就好了。 后端使用 python  或 golang 实现 ， 用户密码保存到文本中即可。如果部署到公有云服务上去运行更好；
3. 在虚拟机中(如 virtualbox，vmware) 安装一个 ubuntu 操作系统，然后在其上面部署一个小网站，如任务2中的 网站；
4. 做1个简单的聊天应用，使用TCP、Websocket 或 UDP 协议均可，要求如下：

    - 一个程序做Server，可以是一个控制台程序，用于消息转发；

    - 一个程序做Client，模拟多个用户，使用 Console 或 GUI技术均可，可以启动多个client实例。比如Client A实例，可以发一个消息给Client B实例。Client B也可以回一个消息给Client A；
5. 开发一个简单安卓app，向互联网某个开放服务获取数据（如天气或其他web service数据） 显示在APP界面上。运行结果截屏；
6. 使用 Threejs 实现一个场景：有一个方块，他能沿着某个路径移动。 最好是可以由外部程序控制，即通过通讯（比如：http，websocket ）等方式将位置、速度信息发送给此3D程序，方块将按照上面给定的速度移动到此位置。
